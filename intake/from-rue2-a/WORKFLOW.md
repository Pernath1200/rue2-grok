# How to save changes to GitHub

When you make changes to the A1 app, save them to GitHub with these steps.

1. Open a terminal in the `rue2-a` folder.

2. Run:

```bash
git add .
git commit -m "Description of what you changed"
git push
```

Replace `"Description of what you changed"` with a short note about your edits (e.g. `"Add more Present Simple questions"` or `"Fix typo in articles curriculum"`).

3. Your changes will be on GitHub and, if you use GitHub Pages, the live site will update within a few minutes.
